Acerca de Game Market

Bienvenido a Game Market, tu plataforma ideal para descubrir y comprar los mejores videojuegos del mercado. Nuestra misión es ofrecer una amplia variedad de juegos para todo tipo de jugadores: desde los más casuales hasta los más apasionados.

Este sitio forma parte de un proyecto universitario desarrollado para aplicar conceptos modernos de desarrollo web con Angular, como enrutamiento, componentes y formularios reactivos.

¿Qué vas a encontrar acá?

  • Una lista seleccionada de videojuegos populares y nuevos
  • Páginas de producto con precios, descripciones y estado de stock
  • Un botón para cambiar entre modo claro y oscuro
  • Consumo de una API + uso de formularios reactivos

Tecnologías utilizadas

  • Angular
  • TypeScript
  • Bootstrap
  • HTML y SCSS

Este sitio fue creado como proyecto de práctica para el seminario de angular 2025.
